Step 2: Installation Instructions (XAMPP)

  1. Download XAMPP from the link above (choose the version for your OS)
  2. Run the installer and follow the setup wizard
  3. Select components: Make sure "Apache" and "MySQL" are checked
  4. Choose installation directory: Default is usually fine (C:\xampp)
  5. Complete the installation and launch XAMPP Control Panel
‚ö†Ô∏è Windows Security: If Windows Firewall asks for permissions, click "Allow access" for both Apache and MySQL.

Step 3: Start MySQL Service

In XAMPP Control Panel:

  1. Click the "Start" button next to Apache
  2. Click the "Start" button next to MySQL
  3. Both should show green background when running
Expected Result:
Apache: Running Port 80, 443
MySQL: Running Port 3306

Step 4: Create Database

Option 1: Using phpMyAdmin (Recommended)

  1. In XAMPP Control Panel, click "Admin" next to MySQL
  2. This opens phpMyAdmin in your browser
  3. Click "New" in the left sidebar
  4. Enter database name: pathlab_pro
  5. Click "Create"

Option 2: Using MySQL Command Line

mysql -u root -p
CREATE DATABASE pathlab_pro;
exit;

üîß Troubleshooting

Common Issues:
  • Check if another program is using port 3306
  • Try running XAMPP as Administrator
  • Check XAMPP logs for specific error messages
  • Temporarily disable antivirus/firewall to test
  • Ensure MySQL service is running (green in XAMPP)
  • Check that port 3306 is not blocked
  • Try connecting to 127.0.0.1 instead of localhost
  • Restart XAMPP services
  • Default XAMPP MySQL has no password for root
  • Try connecting without a password first
  • If you set a password, update config_working.php
  • Reset MySQL root password if needed
